
# React Frameworks Assignment

This repository contains four different React-based applications created to explore and compare modern React frameworks. Each project demonstrates the setup and development workflow of a different framework.



## Projects Included

### 1. Next.js Application
Folder: `my-nextjs-app`

Description:  
A React application built using Next.js to demonstrate server-side rendering and modern React routing.

How to Run:

cd my-nextjs-app
npm install
npm run dev


### 2. Remix Application
Folder: `remix-app`

Description:  
A React application built using Remix to explore nested routing and full-stack React features.

How to Run:

cd remix-app
npm install
npm run dev



### 3. Gatsby Application
Folder: `my-gatsby-site`

Description:  
A static site built with Gatsby to explore static site generation and plugin-based architecture.

How to Run:

cd my-gatsby-site
npm install
npm run develop

### 4. Expo (React Native) Application
Folder: `my-expo-app`

Description:  
A mobile application built with Expo and React Native to explore cross-platform mobile development.

How to Run:

cd my-expo-app
npm install
npx expo start

After running Expo:
- Press `i` to open iOS simulator
- Press `a` to open Android
- Press `w` to open in web browser
